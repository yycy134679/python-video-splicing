# 视频拼接工具（内容运营）使用文档

## 1. 先安装 Python（请自行 AI 搜索安装教程）
请先在你的电脑上安装 Python 3.11+。

建议直接在 AI 工具里搜索：
- `Mac 安装 Python 3.11 教程`

安装完成后，打开终端验证：

```bash
python3 --version
```

如果看到版本号（例如 `Python 3.11.x` / `3.12.x`），说明安装成功。

---

## 2. 下载项目 ZIP 压缩包
1. 打开项目地址：[python-video-splicing](https://github.com/yycy134679/python-video-splicing)
2. 点击 `Code` -> `Download ZIP`
3. 浏览器会下载一个类似 `python-video-splicing-main.zip` 的压缩包

---

## 3. 解压、进入目录、安装依赖、启动

## 3.1 解压到任意位置（示例）
你可以解压到任何你方便找到的位置。

常见示例：
- Mac: `~/Work/`

### Mac 示例
```bash
# 假设 zip 在下载目录
cd ~/Downloads

# 解压到 ~/Work
mkdir -p ~/Work
unzip python-video-splicing-main.zip -d ~/Work
```

解压后项目目录通常是：
- `~/Work/python-video-splicing-main`

---

## 3.2 在终端 `cd` 到项目根目录

### Mac
```bash
cd ~/Work/python-video-splicing-main
```

你可以用下面命令确认自己在根目录（能看到 `app.py` 和 `requirements.txt`）：

```bash
ls
```

---

## 3.3 安装依赖

### Mac
```bash
# 在项目根目录执行
python3 -m venv .venv
source .venv/bin/activate
python -m pip install -r requirements.txt
```

---

## 3.4 启动工具
在项目根目录执行：

```bash
streamlit run app.py
```

启动后，终端会显示本地访问地址（通常是 `http://localhost:8501`），浏览器打开即可使用。

---

## 4. 首次启动常见问题

如果页面提示：
- `未找到 ffmpeg 可执行文件`
- `未找到 ffprobe 可执行文件`

说明机器还没安装 ffmpeg。

### Mac 可直接执行
```bash
brew install ffmpeg
```

安装后重新启动：
```bash
streamlit run app.py
```

---

## 5. 每次使用的最短流程
1. 打开终端，`cd` 到项目根目录
2. 激活虚拟环境
3. 启动 `streamlit run app.py`
4. 浏览器打开页面后填入 `PID` 和 `视频链接`（一行对应一条）

### Mac 快捷命令
```bash
cd ~/Work/python-video-splicing-main
source .venv/bin/activate
streamlit run app.py
```
